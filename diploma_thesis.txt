Структура Сайта:

Главная страница:
- Отображение списка последних статей.
- Поиск по ключевым словам.
- Кнопка добавление статьи.

Страница статьи:
- Заголовок и текст.
- Комментарии к статье.
- Счетчик лайков и дизлайков.
- Кнопка добавления в закладки.

Личный кабинет пользователя:
- Профиль пользователя.
- Список добавлены статей.
- Закладки пользователя
- Редактирование статей.

***************************************************************************************************************************

Возможности Сайта:

Авторизация и Аутентификация:
- Регистрация новых пользователей.
- Вход и выход из системы.

Управление статьями:
- Добавление новых статей.
- Редактирование и удаление своих статей.

Комментирование:
- Добавление комментариев к статьям.
- Возможность ответа на комментарии.

Лайки и Дизлайки:
- Возможность ставить лайки и дизлайки.
- Отображение общего количества лайков и дизлайков.

Закладки:
- Добавление статей в закладки:

Поиск:
- Поиск статей по ключевым словам

************************************************************************************************************
Приложения Django:

accounts:
   - Модели для хранения информации о пользователях.
   - Формы для регистрации, входа и восстановления пароля.
   - Представления для работы с профилями пользователей.

articles:
    - Модели для статей, комментариев, лайков и дизлайков.
   - Формы для добавления и редактирования статей и комментариев.
   - Представления для отображения статей, комментариев и управления ими.

bookmarks:
- Модели для хранения информации о закладках пользователей.
  - Представления для работы с закладками.

search:
 - Представления и функции для реализации поиска.

core:
- Общие шаблоны, статические файлы, базовые представления.

************************************************************************************************************
Технологии:
- Django фреймворк.
- HTML, CSS, JavaScript для визуализации и управления интерфейсом.


Библиотеки:

django-allauth:
 - Предоставляет готовые решения для управления учетными записями пользователей, включая регистрацию, вход и управление профилем.

Pillow:
- Библиотека для обработки изображений

django-tinymce:
 - Обеспечивает интеграцию WYSIWYG-редактора для поля текста в формах.


(Возможно еще пополню  список в процессе разработки.)
